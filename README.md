# Office AV Installation Project

## Role
IT & AV Project Coordinator

## Business Context
A company upgraded its office meeting rooms to support hybrid work. The project included video conferencing, audio systems, and screen sharing tools.

## Project Objective
To implement reliable AV systems for 6 meeting rooms, enabling smooth video calls and presentations.

## Tools Used
- Zoom, MS Teams
- AV Equipment
- Vendor Coordination
- IT Support
- Project Scheduling

## Responsibilities
- Coordinated AV vendors  
- Managed installation schedule  
- Tested audio/video systems  
- Supported users during meetings  
- Documented AV procedures  

## Deliverables
- Room Setup Plan  
- Installation Schedule  
- Testing Checklist  
- Issue Log  
- User Guide  

## Outcome
All meeting rooms were successfully upgraded, improving collaboration and reducing meeting disruptions.
